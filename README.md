**Deskripsi**

Katalog Produk Mini adalah aplikasi web frontend murni yang menerapkan operasi CRUD (Create, Read, Update, Delete) tanpa backend atau API.
Data disimpan menggunakan localStorage, sehingga aplikasi dapat berjalan offline.

**Fitur**

CRUD Produk (Tambah, Lihat, Edit, Hapus)
Pencarian & Filter Produk
Sorting (Nama, Harga, Tanggal)
Rating Produk (1–5)
Dashboard Statistik
Notifikasi Interaktif
Desain Responsif (Desktop, Tablet, Mobile)

**Teknologi**

HTML5
CSS3 (Flexbox & Grid)
JavaScript Vanilla
Font Awesome
Google Fonts (Poppins & Inter)
localStorage

**Cara Menjalankan
**
Clone atau download repository
Buka index.html di browser
Tidak memerlukan server atau instalasi

**Data Produk
**
Aplikasi dilengkapi contoh produk seperti:
Laptop, Smartphone, Pakaian, Buku, Kopi, dan Sepatu.

**⚠️ Catatan
**
Data hanya tersimpan di browser yang sama
Tidak cocok untuk data sensitif
Membutuhkan browser modern dengan JavaScript aktif
